# IEMV

**Intrinsic Equivalent Multi-View Deep Reinforcement Learning** for combinatorial optimization problems:
- Flexible Flow Shop Scheduling (FFSP)
- Asymmetric Traveling Salesman Problem (ATSP)
- Pickup and Delivery Traveling Salesman Problem (PDTSP)

## Training

```bash
python train.py
```

## Testing

```bash
python test.py
```
